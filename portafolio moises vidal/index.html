De Cero a Expertos: Tecnologías Aprendidas en el Curso de Desarrollo Front End

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para crear y estructurar contenido en la web. Define la estructura de las páginas web mediante etiquetas.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, el formato y la apariencia de las páginas web.

Javascrip

JavaScript es un lenguaje de programación utilizado para crear interactividad y dinamismo en las páginas web. Permite manipular el contenido HTML y CSS, responder a eventos y realizar tareas asíncronas.

Vue.js

Vue.js es un framework progresivo de JavaScript utilizado para construir interfaces de usuario y aplicaciones web de una sola página (SPA). Se centra en la vista y permite la creación de componentes reutilizables.
